# Functions to find cliques, i.e. complete subgraphs in a graph

**\[deprecated\]**

`largest.cliques()` was renamed to
[`largest_cliques()`](https://r.igraph.org/reference/cliques.md) to
create a more consistent API.

## Usage

``` r
largest.cliques(graph)
```

## Arguments

- graph:

  The input graph.
