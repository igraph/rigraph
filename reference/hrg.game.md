# Sample from a hierarchical random graph model

**\[deprecated\]**

`hrg.game()` was renamed to
[`sample_hrg()`](https://r.igraph.org/reference/sample_hrg.md) to create
a more consistent API.

## Usage

``` r
hrg.game(hrg)
```

## Arguments

- hrg:

  A hierarchical random graph model.
