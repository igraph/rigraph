# Line graph of a graph

**\[deprecated\]**

`line.graph()` was renamed to
[`make_line_graph()`](https://r.igraph.org/reference/make_line_graph.md)
to create a more consistent API.

## Usage

``` r
line.graph(graph)
```

## Arguments

- graph:

  The input graph, it can be directed or undirected.
