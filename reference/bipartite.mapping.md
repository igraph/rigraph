# Decide whether a graph is bipartite

**\[deprecated\]**

`bipartite.mapping()` was renamed to
[`bipartite_mapping()`](https://r.igraph.org/reference/bipartite_mapping.md)
to create a more consistent API.

## Usage

``` r
bipartite.mapping(graph)
```

## Arguments

- graph:

  The input graph.
