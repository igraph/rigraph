# Independent vertex sets

**\[deprecated\]**

`largest.independent.vertex.sets()` was renamed to
[`largest_ivs()`](https://r.igraph.org/reference/ivs.md) to create a
more consistent API.

## Usage

``` r
largest.independent.vertex.sets(graph)
```

## Arguments

- graph:

  The input graph.
