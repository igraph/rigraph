# Check whether a graph is directed

**\[deprecated\]**

`is.directed()` was renamed to
[`is_directed()`](https://r.igraph.org/reference/is_directed.md) to
create a more consistent API.

## Usage

``` r
is.directed(graph)
```

## Arguments

- graph:

  The input graph
