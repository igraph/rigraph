# Checks whether the graph has a vertex attribute called `type`

**\[deprecated\]**

`is.bipartite()` was renamed to
[`is_bipartite()`](https://r.igraph.org/reference/is_bipartite.md) to
create a more consistent API.

## Usage

``` r
is.bipartite(graph)
```

## Arguments

- graph:

  The input graph
