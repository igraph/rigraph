# Checks whether the graph has a vertex attribute called `type`.

It does not check whether the graph is bipartite in the mathematical
sense. Use
[`bipartite_mapping()`](https://r.igraph.org/reference/bipartite_mapping.md)
for that.

## Usage

``` r
is_bipartite(graph)
```

## Arguments

- graph:

  The input graph
