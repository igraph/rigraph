# A graph with no edges

A graph with no edges

## Usage

``` r
make_empty_graph(n = 0, directed = TRUE)

empty_graph(...)
```

## Arguments

- n:

  Number of vertices.

- directed:

  Whether to create a directed graph.

- ...:

  Passed to `make_graph_empty`.

## Value

An igraph graph.
