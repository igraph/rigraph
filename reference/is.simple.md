# Simple graphs

**\[deprecated\]**

`is.simple()` was renamed to
[`is_simple()`](https://r.igraph.org/reference/simplify.md) to create a
more consistent API.

## Usage

``` r
is.simple(graph)
```

## Arguments

- graph:

  The graph to work on.
