# Interactive plotting of graphs

**\[deprecated\]**

`tkplot.center()` was renamed to
[`tk_center()`](https://r.igraph.org/reference/tkplot.md) to create a
more consistent API.

## Usage

``` r
tkplot.center(tkp.id)
```

## Arguments

- tkp.id:

  The id of the tkplot window to close/reshape/etc.
