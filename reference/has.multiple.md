# Find the multiple or loop edges in a graph

**\[deprecated\]**

`has.multiple()` was renamed to
[`any_multiple()`](https://r.igraph.org/reference/which_multiple.md) to
create a more consistent API.

## Usage

``` r
has.multiple(graph)
```

## Arguments

- graph:

  The input graph.
