# Sample from a hierarchical random graph model

`sample_hrg()` samples a graph from a given hierarchical random graph
model.

## Usage

``` r
sample_hrg(hrg)
```

## Arguments

- hrg:

  A hierarchical random graph model.

## Value

An igraph graph.
