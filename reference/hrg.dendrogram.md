# Create an igraph graph from a hierarchical random graph model

**\[deprecated\]**

`hrg.dendrogram()` was renamed to
[`hrg_tree()`](https://r.igraph.org/reference/hrg_tree.md) to create a
more consistent API.

## Usage

``` r
hrg.dendrogram(hrg)
```

## Arguments

- hrg:

  A hierarchical random graph model.
