# Directed acyclic graphs

**\[deprecated\]**

`is.dag()` was renamed to
[`is_dag()`](https://r.igraph.org/reference/is_dag.md) to create a more
consistent API.

## Usage

``` r
is.dag(graph)
```

## Arguments

- graph:

  The input graph. It may be undirected, in which case `FALSE` is
  reported.
