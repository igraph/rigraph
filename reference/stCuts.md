# List all (s,t)-cuts of a graph

**\[deprecated\]**

`stCuts()` was renamed to
[`st_cuts()`](https://r.igraph.org/reference/st_cuts.md) to create a
more consistent API.

## Usage

``` r
stCuts(graph, source, target)
```

## Arguments

- graph:

  The input graph. It must be directed.

- source:

  The source vertex.

- target:

  The target vertex.
