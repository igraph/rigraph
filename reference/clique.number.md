# Functions to find cliques, i.e. complete subgraphs in a graph

**\[deprecated\]**

`clique.number()` was renamed to
[`clique_num()`](https://r.igraph.org/reference/cliques.md) to create a
more consistent API.

## Usage

``` r
clique.number(graph)
```

## Arguments

- graph:

  The input graph.
