# Independent vertex sets

**\[deprecated\]**

`maximal.independent.vertex.sets()` was renamed to
[`max_ivs()`](https://r.igraph.org/reference/ivs.md) to create a more
consistent API.

## Usage

``` r
maximal.independent.vertex.sets(graph)
```

## Arguments

- graph:

  The input graph.
