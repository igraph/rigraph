# Create a graph from the Graph Atlas

**\[deprecated\]**

`graph.atlas()` was renamed to
[`graph_from_atlas()`](https://r.igraph.org/reference/graph_from_atlas.md)
to create a more consistent API.

## Usage

``` r
graph.atlas(n)
```

## Arguments

- n:

  The id of the graph to create.
