# Dyad census of a graph

**\[deprecated\]**

`dyad.census()` was renamed to
[`dyad_census()`](https://r.igraph.org/reference/dyad_census.md) to
create a more consistent API.

## Usage

``` r
dyad.census(graph)
```

## Arguments

- graph:

  The input graph. A warning is given if it is not directed.
