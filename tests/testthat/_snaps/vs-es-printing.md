# vs printing

    Code
      V(g)[[1]]
    Output
      + 1/3 vertex, named:
        name color weight
      1    A   red     10
    Code
      V(g)[[2]]
    Output
      + 1/3 vertex, named:
        name color weight
      2    B   red      9
    Code
      V(g)[1:2]
    Output
      + 2/3 vertices, named:

    Code
      V(g)[2:3]
    Output
      + 2/3 vertices, named:
      [1] B C
